3 Final Glitch Art Images

Glitch images using Notepad ++ or Audacity.

Notepad ++ Glitches

Use Notepad ++ in order to modify the code of the .jpeg image to create glitches into the picture. I made 2 art pieces utilizing this technique.

Audacity Glitch

Convert the .jpeg file into a .raw file and import it into Audacity. Use the Audacity effects onto the .raw file then export it and convert it back into a .jpeg file.
